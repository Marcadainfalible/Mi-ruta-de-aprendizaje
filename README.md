# Mi Ruta de Aprendizaje

Aplicativo web Flask para registrar fichas de observacion de evaluacion formativa de Educacion Inicial, evaluar estudiantes, calcular resultados y generar formatos imprimibles.

## Ejecutar

```powershell
python -m pip install -r requirements.txt
python seed.py
python run.py
```

Luego abre:

```text
http://127.0.0.1:5000
```

## Usuarios demo

- Administrador: `admin@miruta.local` / `admin123`
- Docente: `docente@miruta.local` / `docente123`

## Modulos

- Login y logout con Flask-Login.
- Roles de administrador y docente.
- Dashboard.
- Registro de fichas, estudiantes y criterios editables.
- Catalogo CNEB interno cargado en SQLite.
- Matriz de evaluacion AD, A, B y C con evidencias.
- Resultados estadisticos y graficos Chart.js.
- Formatos imprimibles A4 horizontal para ficha y reporte.
